# Étude de Réalisation d'une Clinique

Diagramme de Gantt interactif présentant les 9 phases d'une étude de réalisation d'une clinique médicale.

## 🔗 Voir en ligne

Ouvrez le fichier `index.html` ou visitez le site GitHub Pages une fois déployé.

## 📁 Contenu

- `index.html` — Diagramme de Gantt interactif (9 phases, 56 mois)
- `Etude_Realisation_Clinique.pdf` — Document PDF détaillé de toutes les étapes
- `gantt_clinique.html` — Copie originale du Gantt

## 📋 Phases du projet

1. **Programmation & Faisabilité** (2–4 mois)
2. **Esquisse** (1–2 mois)
3. **Avant-Projet Sommaire** (2–3 mois)
4. **Avant-Projet Définitif** (2–3 mois)
5. **Projet PRO** (3–4 mois)
6. **DCE & Consultation** (2–3 mois)
7. **Exécution & Synthèse** (2–3 mois)
8. **Chantier** (18–30 mois)
9. **Réception** (2–4 mois)

**Durée totale estimée : 34 à 56 mois (3 à 5 ans)**
